## What is a task?

A task is a conversation between you and Mira focused on a specific goal. Each task has its own context, conversation history, and generated files. All your tasks are listed in the sidebar under **Tasks**.

## Creating a task

Click **New Task** in the sidebar. You'll see the home screen with:

- A text input area — type your request here
- An **Add photos or files** button — attach images or documents for context
- **Try these scenarios** — four pre-written examples you can click to get started:

```
Full-stack engineer with experience in React & Node, working at a
fast-growing organization; based in Berlin or open to remote work.
```

```
Looking for a senior PM who's shipped AI-powered B2B products —
bonus if they've worked at companies like Notion or Figma.
```

Click **Submit** (or press Enter) to send your message.

## Inside a task

Once a task is created, the task page shows:

### Header bar

- **Task title** — Auto-generated from your first message
- **View all files** — Access all data files generated in this task
- **Task Details** — View task metadata

### Conversation

The main area displays the conversation between you and Mira:

- **Your messages** — Shown with a Copy button
- **Mira's responses** — Include text analysis, data tables, and file attachments

### Mira's workflow steps

When Mira processes your request, you'll see the steps it takes:

1. **People Search** — Mira generates search queries and runs them
2. **People Data Generation** — Results are compiled into structured candidate profiles
3. **Results** — Candidates presented as text summaries, tables, or downloadable files

### Data tables

Mira often presents results in interactive tables with:

| Action | Description |
| --- | --- |
| **Copy table** | Copy the table contents to your clipboard |
| **Download table** | Export the table as a file |
| **View fullscreen** | Expand the table for easier reading |

### File attachments

Generated candidate data files appear as cards showing:
- File name
- Data type (e.g., "People Data")
- Number of candidates (e.g., "30 人")
- **Download file** button

## Managing tasks

### From the sidebar

Each task in the sidebar has a **More** (⋯) menu with:

- **Share** — Share the task with others
- **Rename** — Change the task title
- **Delete** — Permanently remove the task

### Continuing a task

Click any task in the sidebar to reopen it. Type a follow-up message to continue the conversation — Mira retains full context from earlier messages.

### Input options

At the bottom of every task:
- **Text input** — Type your message
- **Add photos or files** — Attach supporting documents
- **Submit** — Send your message
