导航菜单是每个网站上的链接列表。

每次添加新页面时，你都需要更新 `docs.json`。页面不会自动显示。

## 导航语法

导航语法是递归的，你可以创建嵌套的导航组。页面名称中不需要包含 `.mdx`。

<CodeGroup>

```json 基础导航
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": ["quickstart"]
        }
      ]
    }
  ]
}
```

```json 嵌套导航
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": [
            "quickstart",
            {
              "group": "Nested Reference Pages",
              "pages": ["nested-reference-page"]
            }
          ]
        }
      ]
    }
  ]
}
```

</CodeGroup>

## 文件夹

将 MDX 文件放在文件夹中，并更新 `docs.json` 中的路径即可。

例如，要在 `https://yoursite.com/your-folder/your-page` 创建页面，你需要创建一个名为 `your-folder` 的文件夹，其中包含名为 `your-page.mdx` 的 MDX 文件。

<Warning>

不能使用 `api` 作为顶层文件夹名称，除非将其嵌套在另一个文件夹中。Mintlify 使用 Next.js，它会保留顶层 `api` 文件夹用于内部服务器调用。`api-reference` 这样的名称是可以接受的。

</Warning>

```json 带文件夹的导航
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## 隐藏页面

未包含在 `docs.json` 中的 MDX 文件不会显示在侧边栏中，但可以通过搜索栏和直接链接访问。
