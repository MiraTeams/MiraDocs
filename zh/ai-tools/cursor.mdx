使用 Cursor 帮助编写和维护文档。本指南展示如何配置 Cursor 以在技术写作任务和使用 Mintlify 组件方面获得更好的效果。

## 前提条件

- 已安装 Cursor 编辑器
- 可访问文档仓库

## 项目规则

创建所有团队成员都可以使用的项目规则。在文档仓库根目录中：

```bash
mkdir -p .cursor
```

创建 `.cursor/rules.md` 文件，添加适合你的技术写作规范和 Mintlify 组件参考。
