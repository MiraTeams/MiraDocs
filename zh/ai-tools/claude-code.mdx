Claude Code 是 Anthropic 的官方 CLI 工具。本指南将帮助你设置 Claude Code 来编写和维护文档。

## 前提条件

- 有效的 Claude 订阅（Pro、Max 或 API 访问）

## 设置

1. 全局安装 Claude Code：

  ```bash
  npm install -g @anthropic-ai/claude-code
```

2. 进入文档目录。
3. （可选）将下面的 `CLAUDE.md` 文件添加到项目中。
4. 运行 `claude` 启动。

## 创建 `CLAUDE.md`

在文档仓库根目录创建 `CLAUDE.md` 文件，为 Claude Code 提供你的文档标准和项目上下文信息。
