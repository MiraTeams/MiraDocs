## 评估框架

当 Mira 返回搜索结果时，不仅仅是列出名单——它还提供结构化的评估来帮助你快速做出决策。

### 分级推荐

候选人按匹配质量分组：

| 梯队 | 说明 |
| --- | --- |
| **第一梯队** | 极高匹配——在核心需求方面有直接经验，背景优秀 |
| **第二梯队** | 高匹配——与需求高度重合，存在少量差距 |
| **第三梯队** | 中等匹配——具有相关的相邻经验 |

### 逐人分析

对于每位推荐的候选人，Mira 提供：
- **当前职位** — 所在公司和职责
- **背景** — 教育经历、职业发展轨迹、突出成就
- **亮点** — 该候选人为什么适合你的特定职位
- **可用性信号** — 博士毕业时间、近期职位变动等

## 请求更深入的分析

你可以让 Mira 对任意候选人或对比进行更深入的分析：

```
Compare the top 3 candidates side by side.
```

```
Which of these candidates would be easiest to recruit?
```

```
Draft an outreach message for Jiayi Weng highlighting why this role is a good fit.
```

## 优先级推荐

Mira 提供可操作的优先级建议：

- **立即行动** — 已准备就绪、背景最强的候选人
- **储备人才** — 即将毕业的博士或可能对新机会感兴趣的候选人
- **战略目标** — 适合担任顾问或领导角色的资深领袖或研究员
