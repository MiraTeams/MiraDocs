Mira 与你日常使用的工具集成，简化你的招聘工作流程。

## 支持的集成

<CardGroup cols={2}>
  <Card title="Gmail" icon="envelope">
    直接在 Mira 中发送外联邮件并追踪回复。
  </Card>
  <Card title="Slack" icon="hashtag">
    接收通知，与团队分享候选人动态。
  </Card>
  <Card title="Airtable" icon="table">
    将候选人数据同步到你的 Airtable 数据库，便于追踪和协作。
  </Card>
  <Card title="OneDrive" icon="cloud">
    从 OneDrive 存储中访问和附加文件。
  </Card>
</CardGroup>

## Chrome 扩展

Mira Chrome 扩展通过捕获你浏览的网页上的候选人信息来扩展 Mira 的数据覆盖范围。这对于那些未被 Mira 默认数据源索引的平台上的候选人档案尤为有用。

<Card title="Chrome 扩展" icon="puzzle-piece" href="/zh/integrations/chrome-extension" horizontal>
  了解如何安装和使用 Mira Chrome 扩展。
</Card>

## 文件附件

在任何任务中，你都可以使用对话底部的**添加图片或文件**按钮直接附加文件。支持的文件类型包括图片和文档，可为你的搜索提供额外的上下文信息。
