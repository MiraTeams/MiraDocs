## 登录或注册

访问 [mira.day/sign-in](https://mira.day/sign-in) 开始使用。Mira 支持三种登录方式：

<CardGroup cols={3}>
  <Card title="Google" icon="google">
    使用 Google 账户一键登录。
  </Card>
  <Card title="Microsoft" icon="microsoft">
    使用 Microsoft 账户一键登录。
  </Card>
  <Card title="邮箱" icon="envelope">
    输入邮箱地址，按照验证流程完成登录。
  </Card>
</CardGroup>

<Info>
登录和注册使用同一页面。如果你还没有账户，首次登录时系统会自动为你创建。
</Info>

## 你的工作区

登录后，你将进入 [mira.day/task](https://mira.day/task) 的主任务界面。界面主要包括：

- **侧边栏**（左侧）— 任务列表、新建任务按钮和账户菜单
- **主区域**（中央）— 任务对话或包含推荐场景的首页

## 账户设置

点击侧边栏左下角的用户名即可访问以下设置：

| 设置项 | 选项 |
| --- | --- |
| **语言** | English、简体中文 |
| **主题** | 浅色、深色、跟随系统 |
| **退出登录** | 结束当前会话 |

## 退出登录

在账户菜单中选择**退出登录**。你将被重定向到登录页面，随时可以重新登录——你的任务和数据都会被保留。
