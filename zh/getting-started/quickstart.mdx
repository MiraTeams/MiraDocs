## 注册

Mira 目前处于内测阶段。访问 [mira.day](https://mira.day) 加入候补名单，或在已有访问权限时直接登录。

## 你的首次候选人搜索

<Steps>
  <Step title="新建任务">
    点击侧边栏中的**新建任务**按钮，你会看到一个文本输入框，在其中描述你要查找的内容。
  </Step>
  <Step title="描述职位">
    用自然语言描述你的需求，尽可能具体：

    ```
    Looking for a senior full-stack engineer with React and Node.js experience,
    based in Berlin or open to remote work, ideally from a fast-growing startup.
    ```

    <Tip>你提供的背景信息越丰富——技能、地点、资历、行业——Mira 的搜索结果就越精准。</Tip>
  </Step>
  <Step title="查看结果">
    Mira 会跨多个数据源进行搜索，并返回一份结构化的候选人列表。每条结果包括：
    - 姓名和当前职位
    - 背景和资历
    - 与你的需求的匹配分析

    结果可导出为结构化数据文件。
  </Step>
  <Step title="优化搜索">
    继续对话以缩小结果范围：

    ```
    Show me only candidates with published research papers.
    ```

    Mira 会在任务中保持上下文，并据此优化结果。
  </Step>
</Steps>
