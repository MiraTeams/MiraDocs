The Mira Chrome Extension lets Mira access candidate data from web pages you browse, supplementing its built-in data sources.

## Installation

<Steps>
  <Step title="Get the extension">
    Install the Mira Chrome Extension from the Chrome Web Store (or request access from your account manager during beta).
  </Step>
  <Step title="Sign in">
    Click the Mira extension icon in your browser toolbar and sign in with the same account you use on mira.day.
  </Step>
  <Step title="Start using">
    When browsing candidate profiles or relevant web pages, the extension allows Mira to incorporate that data into your active tasks.
  </Step>
</Steps>

## How it works

When you encounter candidate information on the web that Mira doesn't already have, the Chrome Extension can capture structured and unstructured data from the page and make it available within your tasks.

This is useful for:
- Professional profiles on niche platforms
- Company team pages
- Conference speaker lists
- Academic publication pages
