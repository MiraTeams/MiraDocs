## Chrome Extension

The Mira Chrome Extension extends Mira's data coverage by allowing it to capture candidate information from web pages you visit. This is especially useful for profiles on platforms that aren't indexed by Mira's default data sources.

  Learn how to install and use the Mira Chrome Extension.
</Card>

## File attachments

Within any task, you can attach files directly using the **Add photos or files** button at the bottom of the conversation. Supported file types include images and documents that provide additional context for your search.
